# Calculadora

## Requisitos

### Descrição
O sistema consiste em um aplicativo de calculadora desenvolvido para realizar operações matemáticas básicas de forma rápida e intuitiva.

### Objetivo
Permitir que o usuário realize operações matemáticas básicas de maneira simples, rápida e eficiente.

### Requisitos Funcionais
O sistema deve permitir inserir números de 0 a 9.
O sistema deve permitir realizar as seguintes operações:
Adição (+)
Subtração (−)
Multiplicação (×)
Divisão (/)
O sistema deve exibir o resultado da operação ao pressionar o botão "=".
O sistema deve permitir limpar os dados inseridos (botão "C").

### Requisitos Não Funcionais
O sistema deve ter interface simples e intuitiva.
O tempo de resposta ao pressionar um botão deve ser imediato.

### Escopo
O projeto é capaz de realizar operações matematicas básicas (adição, subtração, multiplicação e divisão), por meio de uma interface simples e intuitiva. O usuario inseri os valor numéricos depois seleciona a operação que deseja e visualiza o resultado na tela.
